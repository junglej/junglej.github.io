Leetcode54. Spiral Matrix

题目描述

思路

有四个方向right, left, top, bottom.

  1. direction=”right”， 当到达最右边界时，代表本行输出结束，top++，
  2. direction=”bottom”，当到达最下边界时, right–,
  3. direction = “left”，当到达最左边界是 bottom–,
  4. direction=”top”，当到达top时候，left++，direction=”right”.
